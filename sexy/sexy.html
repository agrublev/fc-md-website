Start

Will start running web app or fkit playground or fkit standalone app

start:web

Start web application will build fkit if it was never built once, but will not rebuild fkit

yarn fsr run-s start:web:open start:web:run

start:playground

Start fkit component playground and docs

yarn fsr run-p fkit:gencomps start:playground:run fkit:serve playground:open

start:fkit:app

Start testing app for fkit Look into StandaloneApp.js

yarn fsr run-p start:fkit:app:run start:fkit:open

Build

Build and deploy

build:web

Build the web application will first build fkit

yarn fsr run build:web:run

build:web:stats

Build the web application will first build fkit with stats

bolt w @f/web build:stats

build:fkit

Builds the release version of fkit

bolt w @f/fkit build

build:playground

Build component playground need to de before deploys!

bolt w @f/fkit build:playground

Deploy

Deploying our websites live

deploy:playground

Deploy fkit to firebase https://freedcamp.design

yarn fsr run-s build:fkit:bump build:playground build:fkit:deploy

deploy:docs

Releases new version from Readme.md and fscripts.md into firebase hosting

yarn fsr run-s gen:docs deploy:docs:fb

create:branch

Create a new branch (so you don't commit to dev)

node packages/scripts/branching/index.js

pull:request

Create a new pull request

node packages/scripts/branching/pullRequest.js

Automation

Generate components and add/remove dependencies

generate:component

Generates a sample component with or without mobx

node packages/scripts/templateGenerator/index.js

generate:page

Generate fkit playground page

node packages/scripts/fkitPageGenerator/index.js

generate:proptypes

npx ts-to-proptypes

dep:find:unused

Finds any unused dependencies

node packages/scripts/findUnusedDeps.js

dep

Installs a new dependency to chosen packages

yarn dep

remove

Removes a dependency from chosen packages

node packages/scripts/installPackage.js remove

Testing

Allows you to run all the different checks to make sure your code is 100%

test

Runs all tests inside the tests package

yarn test

test:watch

Run and keep watching tests

yarn test:watch

test:update:snapshots

Runs and updates snapshots of rootStore

yarn test -u

test:majic

Runs a beautiful web ui to see tests run and coverage

npx majestic

test:clear

Run tests, but clear cache first

yarn test:clear

test:eslint

Test that all code is passing eslint standards

yarn test:eslint

test:eslint:fix

Helps you see ALL warnings and tries to fix them for you

yarn lint

test:coverage

Generate coverage reports then open reports in web browser server

yarn test:coverage

test:coverage:run

Generate coverage reports

yarn test:coverage:run

test:coverage:web

Open web server with coverage reports

yarn test:coverage:web

lint:blame

Test that all code is passing eslint standards and see who contributed bad code :)

yarn lint:blame

lint:generate:json

yarn lint:generate:json

playground:open

Opens the fkit playground browser

const openBrowser = require("./packages/scripts/utils/openBrowser/openBrowser.js");
const fetch = require("node-fetch");

(async () => {
    await new Promise(resolve => setTimeout(resolve, 3500));
    await new Promise(resolve =>
        fetch("http://127.0.0.1:12341/bundle.js")
            .then(res => res.text())
            .then(body => {
                resolve();
            })
    );
    openBrowser("http://0.0.0.0:1234");
})();
